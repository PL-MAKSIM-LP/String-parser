A string is sold for input in format:  3[a]
this mean that result string will be: aaa  

    more examples:  
        a3[5[b]] -> abbbbbbbbbbbbbbb
        2[b]4[a]0[q]1[z] -> bbaaaaz


A string is sold always valid:

    Valid formats:
        2[b], 232[3[1[b]]4[a]]
  
    Non valid formats:
        [3[a]], [b][a], asd[a];
